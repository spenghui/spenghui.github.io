安装

0. 环境

Ubuntu12 32位（其他版本均蛋疼）真机

1. bladerf驱动

$ sudo add-apt-repository -y ppa:git-core/ppa
$ sudo add-apt-repository -y ppa:chris-lea/zeromq
$ sudo add-apt-repository -y ppa:chris-lea/libsodium
$ sudo add-apt-repository -y ppa:ettusresearch/uhd
$ sudo add-apt-repository ppa:bladerf/bladerf
$ sudo apt-get update
$ sudo apt-get -y install git autoconf automake libtool debhelper dpkg-dev sqlite3 libsqlite3-dev g++ libusb-1.0-0-dev
$ sudo apt-get -y install libortp-dev libortp8 libosip2-dev libreadline-dev libncurses5-dev libgsm1-dev cdbs libsqlite0-dev
$ sudo apt-get -y install unixodbc unixodbc-dev libssl-dev libsrtp0-dev libsqliteodbc uuid-dev libjansson-dev libxml2-dev
$ sudo apt-get -y install libboost1.48-all-dev libzmq3-dev libzmq3 python-zmq libsodium13 bladerf libbladerf-dev
$ sudo apt-get -y install libuhd-dev libuhd003 uhd-host
$ sudo apt-get install bladerf-firmware-fx3
$ sudo apt-get install bladerf-fpga-hostedx40   # for the 40 kLE hardware
$ sudo apt-get autoremove

2. LTE-Cell-Scanner

$ git clone https://github.com/JiaoXianjun/LTE-Cell-Scanner.git
$ mkdir build
$ cd build
$ cmake ../ -DUSE_BLADERF=1
$ make

安装中可能存在的问题及解决办法：

-- Try to find BLADERF LIB
-- BLADERF LIB not found
$ gedit cmake/Modules/FindBLADERF.cmake
把/usr/lib/x86_64-linux-gnu/改成/usr/lib/i386-linux-gnu/

cmake - Could not find ITPP library
$ sudo apt-get install libitpp-dev

CMake is not able to find BOOST libraries
$ sudo apt-get install cmake libblkid-dev e2fslibs-dev libboost-all-dev libaudit-dev

Can't fine Boost and FFTW3
$ sudo apt-get install libfftw3-dev
